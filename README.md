# KI-Angebots-Assistent

Next.js Demo fuer Kuechenstudios: technische Stuecklisten hochladen, Positionen erkennen und daraus Kundensprache, Angebotstext und Begleit-E-Mail generieren.

## Lokal starten

```bash
npm install
npm run dev
```

## Environment

Lege eine `.env.local` an:

```bash
OPENAI_API_KEY=dein_key
```
